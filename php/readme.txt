在dist外修改後會送進dist裡使用

需要使用的話請將開發環境移至 xampp/htdocs/
  1. Sourcetree 路徑記得修改

  2. 阿帕契要開著

  3. 路徑範例
     http://localhost/[專案路徑]

使用注意事項
  1. 請不要使用 live server 開網頁

  2. 開發時可以用 gulp 追蹤，修改後網頁要手動重新整理

  3. HTML 檔不會放在這裡 修改過後會移除~

  4. PDO 資料夾裡放的是連線設定 須個別修改!!!

  5. PHP 資料夾之後上線前應該是會改位置!!! 打包進來只是方便測試

  6. fetch 方式可參考 exhibition_information.html
